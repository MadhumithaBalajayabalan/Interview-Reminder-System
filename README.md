# Interview Reminder System

## Project Overview

This Business Analyst case study focuses on analysing interview attendance patterns, identifying recruitment process inefficiencies, and providing data-driven recommendations to reduce interview no-shows. The project combines SQL-based analysis and Power BI reporting to monitor key recruitment metrics and support HR decision-making.

---

## Business Problem

Frequent interview no-shows and rescheduled interviews were impacting recruiter productivity, increasing hiring timelines, and reducing recruitment efficiency. HR stakeholders required visibility into attendance trends, recruiter workload, and departmental hiring activity to improve candidate engagement and recruitment outcomes.

---

## Objectives

* Monitor interview attendance and no-show rates.
* Analyse recruiter performance and workload distribution.
* Track interview rescheduling trends.
* Evaluate departmental hiring activity.
* Provide actionable insights through interactive reporting.

## SQL Analysis

Developed 7 SQL queries to analyse interview attendance, no-show counts, rescheduled interviews, recruiter performance, departmental hiring activity, and overall recruitment KPIs.

### Key Queries

* Total Interviews Conducted
* Attendance Rate Analysis
* No-Show Count
* Rescheduled Interview Analysis
* Interview Distribution by Department
* Recruiter Workload Analysis
* Interview Data Validation and Review
